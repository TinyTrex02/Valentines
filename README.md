# Valentine's Memory Game

A romantic memory card game in a heart shape. Match all pairs to reveal a special Valentine's proposal with fireworks!

## Live Demo

https://valentines-zeta-five.vercel.app/



Password ( top right ) - iloveyou

## How to Use

1. Fork this repo
2. Replace the images in `/public` with your own photos
3. Deploy to Vercel (or any hosting) and share the link with your valentine!

## Features

- Interactive memory card game in a heart shape layout
- Beautiful animations using Framer Motion
- Customizable with your own photos
- Playful "No" button that runs away
- Fireworks animation on acceptance
- Password skip for testing
